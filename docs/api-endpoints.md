# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Questions

- `GET /questions` with answers

### UserQuestions

- `GET /userquestions`
- `POST /userquestion`
- `PATCH /userquestion`
- `DELETE /userquestion`

### Messages

- `GET /messages`
- `POST /message`


### Likes

- `GET /likes`
- `POST /like`
- `DELETE /like`
